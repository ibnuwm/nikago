# CS-999 - Definition of Done

## Sebuah fitur dinyatakan selesai apabila memenuhi seluruh kriteria berikut.

---

## Development

- Requirement sesuai PRD.
- Database sesuai Database Design.
- API sesuai API Specification.
- Mengikuti Coding Standard.
- Tidak ada error pada aplikasi.

---

## Code Quality

- Laravel Pint lulus.
- PHPStan lulus.
- Tidak ada debug code.
- Tidak ada commented code.

---

## Testing

- Feature Test dibuat.
- Semua test lulus.
- Bug regression telah diuji.

---

## Documentation

- Dokumentasi diperbarui bila diperlukan.
- API diperbarui bila ada perubahan.

---

## Git

- Menggunakan Conventional Commit.
- Pull Request telah dibuat.
- Code Review selesai.
- Tidak ada conflict.

---

## Security

- Authorization telah diperiksa.
- Validation lengkap.
- Tidak ada data sensitif pada response.

---

## Performance

- Tidak terdapat N+1 Query.
- Pagination diterapkan pada endpoint list.
- Queue digunakan untuk proses berat.

---

## Deployment

- Siap di-deploy ke environment Staging.
- Tidak ada konfigurasi manual yang belum terdokumentasi.

---

## Checklist

- [ ] Requirement selesai
- [ ] Database selesai
- [ ] API selesai
- [ ] Coding selesai
- [ ] Testing selesai
- [ ] Review selesai
- [ ] Siap Deploy
