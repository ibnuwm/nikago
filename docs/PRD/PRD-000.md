# PRD-000 - Project Overview

## Tujuan

Membangun **Nikago**, sebuah Wedding Super App yang mengintegrasikan seluruh kebutuhan pernikahan dalam satu platform, mulai dari pembuatan website undangan, wedding planner, vendor directory, marketplace, CRM, hingga AI Wedding Assistant.

---

## Business Value

- Menjadi platform terpusat bagi industri pernikahan Indonesia.
- Mengurangi kebutuhan pengguna menggunakan banyak aplikasi berbeda.
- Meningkatkan pendapatan melalui Subscription, Marketplace, AI Credit, dan Premium Services.
- Menjadi fondasi pengembangan Wedding ERP dan Mobile App.

---

## User Story

### Sebagai Calon Pengantin

Saya ingin mengelola seluruh kebutuhan pernikahan dalam satu aplikasi sehingga proses persiapan menjadi lebih mudah.

### Sebagai Vendor

Saya ingin mempromosikan layanan saya dan mendapatkan calon pelanggan baru.

### Sebagai Wedding Organizer

Saya ingin mengelola seluruh project wedding klien dalam satu sistem.

### Sebagai Admin

Saya ingin memonitor seluruh aktivitas platform dari satu dashboard.

---

## Acceptance Criteria

- Platform mendukung Multi Tenant.
- Seluruh modul saling terintegrasi.
- Mendukung Subscription.
- Mendukung Marketplace.
- Mendukung AI.
- Mendukung Vendor Management.
- Mendukung Website Invitation.
- Mendukung Wedding Planner.
