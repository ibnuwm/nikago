# PRD-017 - Vendor Booking

## Tujuan

Memungkinkan pengguna melakukan pemesanan vendor secara online melalui Nikago.

---

## Business Value

- Menjadi sumber pendapatan dari Booking Fee.
- Mempermudah proses reservasi vendor.
- Mengurangi proses booking manual melalui WhatsApp.

---

## User Story

Sebagai calon pengantin, saya ingin melakukan booking vendor secara online sehingga proses pemesanan lebih mudah dan terdokumentasi.

Sebagai vendor, saya ingin menerima dan mengelola booking melalui dashboard.

---

## Acceptance Criteria

- Booking Vendor.
- Pilih Paket.
- Pilih Tanggal.
- Validasi Jadwal.
- Status Booking.
- Invoice.
- Pembayaran.
- Riwayat Booking.
- Cancel Booking.
- Reschedule Booking.
