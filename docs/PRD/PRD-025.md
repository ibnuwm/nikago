# PRD-025 - Notification

## Tujuan

Menyediakan sistem notifikasi terpusat untuk seluruh aktivitas platform.

---

## Business Value

- Meningkatkan engagement pengguna.
- Mengurangi missed reminder.
- Mendukung otomatisasi sistem.

---

## User Story

Sebagai pengguna, saya ingin menerima notifikasi ketika terdapat aktivitas penting sehingga saya tidak melewatkan informasi.

---

## Acceptance Criteria

- In App Notification.
- Email Notification.
- WhatsApp Notification.
- Push Notification (Future).
- Reminder.
- Read / Unread.
- Notification History.
- Notification Preference.

---

## Prioritas

🟠 High

---

## Dependency

- Authentication
- Email Service
- WhatsApp API

---

## Future Improvement

- AI Smart Notification.
- Notification Scheduling.
- Automation Workflow.
- User Preference AI.
