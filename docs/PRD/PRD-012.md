# PRD-012 - Guest Seating

## Tujuan

Membantu pasangan menyusun posisi tempat duduk tamu secara terstruktur.

---

## Business Value

- Mengurangi kesalahan penempatan tamu.
- Mempermudah koordinasi saat acara.
- Menjadi nilai tambah paket Premium.

---

## User Story

Sebagai calon pengantin, saya ingin mengatur posisi duduk tamu sehingga keluarga dan kerabat dapat ditempatkan dengan baik.

---

## Acceptance Criteria

- Membuat daftar meja.
- Menentukan kapasitas meja.
- Memindahkan tamu ke meja tertentu.
- Validasi kapasitas meja.
- Menampilkan daftar tamu per meja.
- Export seating plan.
