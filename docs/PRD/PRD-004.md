# PRD-004 - Website Invitation

## Tujuan

Memungkinkan pengguna membuat website undangan digital dengan mudah tanpa kemampuan teknis.

---

## Business Value

- Menjadi produk utama Nikago.
- Menarik pengguna baru.
- Meningkatkan konversi ke paket Premium.

---

## User Story

Sebagai calon pengantin, saya ingin membuat website undangan sehingga dapat membagikan informasi pernikahan kepada tamu secara online.

---

## Acceptance Criteria

- Membuat website baru.
- Memilih template.
- Mengubah informasi acara.
- Upload foto.
- Countdown.
- Maps.
- Music.
- Gallery.
- Love Story.
- Gift.
- Live Streaming.
- Publish Website.

---

## Prioritas

🔴 Critical

---

## Dependency

- Authentication
- Dashboard
- Template Builder

---

## Future Improvement

- AI Generate Content
- White Label
- Custom CSS
