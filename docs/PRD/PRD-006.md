# PRD-006 - RSVP

## Tujuan

Menyediakan sistem RSVP (Reservation) untuk membantu pasangan mengetahui jumlah tamu yang akan hadir secara real-time.

---

## Business Value

- Mengurangi ketidakpastian jumlah tamu.
- Membantu estimasi konsumsi dan kapasitas venue.
- Menjadi sumber data Wedding Planner.

---

## User Story

Sebagai tamu undangan, saya ingin mengonfirmasi kehadiran saya sehingga pengantin mengetahui apakah saya akan hadir.

Sebagai calon pengantin, saya ingin melihat jumlah tamu yang hadir agar dapat mempersiapkan acara dengan lebih baik.

---

## Acceptance Criteria

- Tamu dapat memilih Hadir / Tidak Hadir.
- Tamu dapat mengisi jumlah pendamping.
- Tamu dapat memberikan ucapan.
- RSVP dapat diperbarui sebelum batas waktu.
- Status RSVP tersimpan secara real-time.
- Statistik RSVP tersedia pada Dashboard.
