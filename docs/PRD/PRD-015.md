# PRD-015 - Vendor Profile

## Tujuan

Memberikan halaman profil lengkap bagi vendor untuk menampilkan informasi bisnis dan layanan.

---

## Business Value

- Meningkatkan kepercayaan calon pelanggan.
- Menjadi identitas digital vendor.
- Mempermudah proses pencarian vendor.

---

## User Story

Sebagai vendor, saya ingin memiliki halaman profil profesional sehingga calon pelanggan dapat melihat layanan yang saya tawarkan.

---

## Acceptance Criteria

- Informasi vendor.
- Logo.
- Cover.
- Deskripsi.
- Portfolio.
- Paket layanan.
- Galeri.
- Lokasi.
- Kontak.
- Jam operasional.
- Media sosial.
