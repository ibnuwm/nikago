# PRD-007 - Guest Management

## Tujuan

Memudahkan pengelolaan daftar tamu mulai dari impor data hingga monitoring kehadiran.

---

## Business Value

- Mengurangi pekerjaan manual.
- Mempermudah distribusi undangan.
- Menjadi sumber data untuk RSVP dan Seating.

---

## User Story

Sebagai calon pengantin, saya ingin mengelola daftar tamu sehingga semua undangan dapat dikirim dengan mudah.

---

## Acceptance Criteria

- Tambah tamu.
- Edit tamu.
- Hapus tamu.
- Import Excel/CSV.
- Export Excel.
- Kelompokkan tamu berdasarkan kategori.
- Cari tamu.
- Filter tamu.
- Lihat status RSVP.
- Riwayat pengiriman undangan.

---

## Prioritas

🔴 Critical

## Dependency

- Authentication
- Website Invitation
