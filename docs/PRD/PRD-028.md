# PRD-028 - Analytics

## Tujuan

Menyediakan dashboard analitik untuk memonitor performa website, pengguna, vendor, transaksi, dan bisnis.

---

## Business Value

- Membantu pengambilan keputusan.
- Monitoring pertumbuhan bisnis.
- Mengukur efektivitas marketing.
- Menjadi nilai tambah paket Business.

---

## User Story

### Sebagai Admin

Saya ingin melihat performa platform sehingga dapat mengambil keputusan berdasarkan data.

### Sebagai Vendor

Saya ingin mengetahui performa profil vendor saya.

---

## Acceptance Criteria

- Dashboard Analytics.
- Visitor Statistics.
- RSVP Statistics.
- Revenue Report.
- Subscription Report.
- Vendor Performance.
- AI Usage Report.
- Export Report.
- Filter Periode.
- Grafik.

---

## Prioritas

🟠 High

---

## Dependency

- Dashboard
- Payment
- Subscription

---

## Future Improvement

- AI Insight.
- Predictive Analytics.
