# PRD-022 - CMS

## Tujuan

Menyediakan Content Management System untuk mengelola seluruh konten website tanpa perlu mengubah kode aplikasi.

---

## Business Value

- Mempermudah tim Marketing.
- Mengurangi ketergantungan terhadap Developer.
- Mempercepat proses publikasi konten.

---

## User Story

Sebagai Admin, saya ingin mengubah konten website sehingga informasi selalu terbaru.

---

## Acceptance Criteria

- Landing Page Management.
- Static Page.
- FAQ.
- Banner.
- Promo.
- Homepage Section.
- Navigation Menu.
- Footer.
- Media Library.
- Draft & Publish.

---

## Prioritas

🟠 High
