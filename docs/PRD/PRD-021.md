# PRD-021 - Admin Panel

## Tujuan

Menyediakan pusat kontrol utama untuk mengelola seluruh operasional platform Nikago, termasuk pengguna, vendor, subscription, transaksi, konten, AI usage, serta monitoring sistem.

---

## Business Value

- Mempermudah pengelolaan platform.
- Mempermudah Customer Service menangani pengguna.
- Mempermudah monitoring revenue dan subscription.
- Menjadi pusat administrasi seluruh modul.

---

## User Story

### Sebagai Super Admin

Saya ingin mengelola seluruh data platform sehingga operasional Nikago dapat berjalan dengan baik.

### Sebagai Customer Service

Saya ingin membantu pengguna ketika mengalami kendala.

### Sebagai Finance

Saya ingin melihat seluruh transaksi dan subscription.

---

## Acceptance Criteria

- Dashboard Admin.
- User Management.
- Vendor Management.
- Subscription Management.
- Payment Monitoring.
- Revenue Dashboard.
- AI Usage Monitoring.
- Activity Log.
- Audit Log.
- System Setting.
- Role & Permission.
- Export Data.
