# PRD-020 - Subscription

## Tujuan

Mengelola paket berlangganan pengguna dan vendor sesuai fitur yang tersedia.

---

## Business Value

- Menjadi sumber pendapatan utama Nikago.
- Mengontrol akses fitur berdasarkan paket.
- Meningkatkan Customer Lifetime Value (CLV).

---

## User Story

Sebagai pengguna, saya ingin memilih paket langganan sehingga dapat menggunakan fitur sesuai kebutuhan saya.

Sebagai Admin, saya ingin mengelola seluruh paket subscription.

---

## Acceptance Criteria

- Paket Free.
- Paket Basic.
- Paket Pro.
- Paket Business.
- Upgrade Paket.
- Downgrade Paket.
- Perpanjang Langganan.
- Riwayat Subscription.
- Auto Renewal.
- Expired Notification.

---

## Prioritas

🔴 Critical
