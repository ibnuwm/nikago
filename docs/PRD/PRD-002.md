# PRD-002 - Dashboard

## Tujuan

Menyediakan halaman utama yang menampilkan ringkasan aktivitas pengguna sesuai role.

---

## Business Value

- Mempermudah navigasi.
- Menampilkan informasi penting secara cepat.
- Meningkatkan engagement pengguna.

---

## User Story

Sebagai Bride, saya ingin melihat progress persiapan pernikahan.

Sebagai Vendor, saya ingin melihat booking terbaru.

Sebagai Admin, saya ingin melihat statistik platform.

---

## Acceptance Criteria

- Dashboard berbeda sesuai role.
- Menampilkan statistik.
- Menampilkan shortcut menu.
- Menampilkan aktivitas terbaru.
- Responsive.

---

## Prioritas

🔴 Critical

- Authentication
