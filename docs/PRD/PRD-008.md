# PRD-008 - Wedding Planner

## Tujuan

Menyediakan pusat manajemen seluruh persiapan pernikahan dalam satu dashboard.

---

## Business Value

- Menjadi pembeda utama dibanding website undangan biasa.
- Meningkatkan retensi pengguna.
- Menambah nilai paket Pro.

---

## User Story

Sebagai calon pengantin, saya ingin mengelola seluruh persiapan pernikahan dalam satu aplikasi sehingga tidak perlu menggunakan banyak tools.

---

## Acceptance Criteria

- Dashboard Planner.
- Checklist.
- Timeline.
- Budget.
- Vendor.
- Reminder.
- Notes.
- Progress Persiapan.

---

🔴 Critical
